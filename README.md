# Atividade_de_conceitos_basicos_liguagem_C
Projeto em Linguagem C com menu interativo que reúne exercícios de lógica e fundamentos da programação. Aplica estruturas condicionais (if/else, switch case) e de repetição (while, do while) para calcular média acadêmica, descontos, fatorial, Fibonacci e P.A. Foco em organização, controle de fluxo e prática da base da programação estruturada.
